Ruta http://localhost:3000:
Al ingresar a esta dirección, el servidor responde con el mensaje “Servidor activo”,
 indicando que está funcionando correctamente.

Ruta http://localhost:3000/saludo:
Esta ruta muestra el mensaje “Hola comunidad” cuando el usuario accede desde el navegador.

Ruta http://localhost:3000/mensaje/juan:
En esta ruta el servidor toma el nombre que se escribe en la dirección (en este caso “juan”)
 y lo usa para responder con un saludo personalizado, mostrando “Hola juan”.
